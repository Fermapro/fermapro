⚙️ Configuration Google Sheets
Connectez l'application à votre Google Sheet partagé pour synchroniser le stock en temps réel avec toute l'équipe.
1
Créez le Google Sheet
Ouvrez Google Drive → Nouveau → Google Sheets. Nommez-le "Fermapro - Stock" et partagez-le avec votre équipe.
2
Ajoutez le script Apps Script
Dans le Sheet → Extensions → Apps Script → copiez-collez le contenu du fichier apps-script.gs → Enregistrez.
3
Déployez le script
Apps Script → Déployer → Nouveau déploiement → Type : Application Web → Accès : Tout le monde → Déployer. Copiez l'URL.
4
Collez l'URL ici
Copiez l'URL de déploiement (commence par https://script.google.com/macros/s/...)
